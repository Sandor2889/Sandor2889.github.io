# Solution

- a에 대한 집합을 set에 저장한다. <br>
- b에 대한 요소를 입력 받을 때 동시에 a와의 공통되는 요소가 있는지 검사한다. <br>
- 공통된 요소가 n개라면 전체 원소의 갯수 (a + b)에서 -2n을 한다. () <br>

***

# Cord
```c++
#include <iostream>
#include <set>

using namespace std;

set<int> sA;

int main()
{
	int a, b, num;
	cin >> a >> b;
	int count = a + b;

	// A 집합 입력
	for (int idx = 0; idx < a; idx++)
	{
		cin >> num;
		sA.insert(num);
	}

	// B 집합 입력 과 동시에 공통된 원소 찾기
	for (int idx = 0; idx < b; idx++)
	{
		cin >> num;
		
		if (sA.find(num) != sA.end())
		{
			count -= 2;
		}
	}

	// 출력
	cout << count;

	return 0;
}
```
