# Solution
- 에라토스테네스의 체를 활용 한다.
- bool을 리턴하는 메서드를 만들어 n까지의 소수만 구한다.
- math 라이브러리를 추가하여 sqrt()를 사용하는 것보다 i를 제곱하였다.

***

# Cord
```c++
#include <string>
#include <vector>

using namespace std;

bool primeNumSeive(int n)
{
    for (int i = 2; i * i <= n; i++)
    {
        if (n % i == 0) { return false; }
    }

    return true;
}

int solution(int n)
{
    int answer = 0;

    for (int i = 2; i <= n; i++)
    {
        if (primeNumSeive(i)) { answer++; }
    }

    return answer;
}
```
