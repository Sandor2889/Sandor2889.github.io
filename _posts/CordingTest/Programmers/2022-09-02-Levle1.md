# Solution
- int prev 이전 숫자를 저장할 변수를 선언하고 for문으로 비교한다.
- 서로 다른 숫자면 vector에 넣으면 된다.

***

# Cord
```c++
#include <vector>

using namespace std;

vector<int> answer;

vector<int> solution(vector<int> arr) 
{   
    int prev = -1;
    for(int i = 0; i < arr.size(); i++)
    {
       if (prev != arr[i])
       {             
            answer.push_back(arr[i]);
       }
       prev = arr[i];
    }

    return answer;
}
```
