# Solution
- 5와 3의 배수로 먼저 탐색을 한다
- 5와 3의 배수가 아니면 모두 -1이라는 값이 나온다
- 5의 배수인지 확인하고 아니라면 3의 배수로 확인한 뒤 결과를 내면 될것같다. <br>
***

# Cord
###  [C#]

```c#
using System;

class Program
{
    static void Main()
    {
        int input = int.Parse(Console.ReadLine());
        int result = 0;

        while(input > 0)
        {
            if (input % 5 == 0)
            {
                input -= 5;
                result++;
            }
            else if (input % 3 == 0)
            {
                input -= 3;
                result++;
            }
            else if (input > 5)
            {
                input -= 5;
                result++;
            }
            else
            {
                result = -1;
                break;
            }
        }
        Console.WriteLine(result);
    }
}
```
