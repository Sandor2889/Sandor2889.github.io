# Solution
- 입력 받은 숫자의 각 자릿수를 vector에 저장한다
- vector를 내림차순으로 정렬하기 위해 bool 매소드를 만들어 정렬
- 정렬된 vector를 출력

# Cord
```c++
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

vector<int> vNum;

bool compare(int _a, int _b)
{
	return _a > _b;
}

int main()
{
	// 입력
	int num;
	cin >> num;

	// 각 자릿수를 vector에 저장
	while (num > 0)
	{
		int units = num % 10;
		vNum.push_back(units);
		num /= 10;
	}

	// 내림차순 정렬
	sort(vNum.begin(), vNum.end(), compare);

	// 출력
	for (int idx : vNum)
	{
		cout << idx;
	}

	return 0;
}
```
